# 安楽死ディベート：攻めの質疑リスト（肯定側）

> 用途：**環節4＝肯定1（あなた）→ 否定1** の質疑。**環節8＝肯定2→否定2** にも流用可。
> 目的：**議論に勝つことではなく、譲歩（admission）を相手の口から引き出すこと。** 取った譲歩はその場で論じず、**反駁・比較衡量で使う**。

## 質疑の鉄則（攻め）— 6つ

1. **閉じた質問だけ**。「はい／いいえ」か二択。理由を聞く「なぜ」は禁止（相手に演説させる）。
2. **一問一事実**。短く。長い前置きをつけない。
3. **譲歩を取ったら、すぐ次へ**。「ありがとうございます」で切り上げ、そこで議論しない。
4. **逃げたら、同じ質問を一度だけ繰り返す**。「はい か いいえ で、お願いします」「短くお願いします」。
5. **答えが自分を利する質問だけ聞く**。相手に有利な答えがありうる質問はしない。
6. **本番は2〜3本に絞る**。否定1の**中心論点を叩く漏斗**を先頭に。⑤の「尊厳死の一貫性」は相手の立場を問わずほぼ常に効くので常備。

## ★相手の実際の4論点への攻め質問【今回の試合・最優先】

> 入手した否定立論（→ `06`）専用。相手の主柱は論点1・3・4の「圧力→自由意思でない」クラスター。下を先に回す。詳しい崩しは 06。
> **通底のコツ**：各論点で「Q＝要件確認」を必ず一本入れ、「圧力動機（迷惑・費用・資源）は我方の要件に入らない」を取る。→ 相手は"無規制の安楽死"を攻撃しているだけ、と示せる。

### 対・論点1（本当の自由意思ではない／社会が先）⚠️敬意を込めて穏やかに
**取りたい譲歩**：尊厳死は認める＝同じ圧力でも禁止しない／我方の要件は身体的苦痛で"迷惑・費用"は要件でない／支援は我方も前置（偽の二択）。
- **Q1**「否定側は、延命治療を拒否して自然な死を選ぶ尊厳死は、認めますか。」→ **はい**ならQ2へ。
- **Q2**「その尊厳死でも、『家族に迷惑をかけたくない』という圧力は働きえますね。」→ **はい**＝取った（圧力は尊厳死にもある＝禁止理由にならない）。
- **Q3**「我方プランの要件は『治療抵抗性の身体的苦痛』です。『家族への迷惑』や『費用』は、要件に入っていますか。」→ **入っていない**＝取った（圧力動機は対象外）。
- **Q4**「我方は緩和ケアなどの支援を尽くすことを前提にしています。『支援を充実させること』に、否定側は反対ですか。」→ **いいえ**＝取った（「支援か死か」は偽の二択、両方やる）。
- **落とし所**：圧力は禁止でなく設計で対処／圧力動機は要件で排除／支援も前置。

### 対・論点2（誤診・不可逆）
- **Q1**「『回復の見込みがない』という診断は、いま現に、尊厳死で治療を中止する根拠として使われていますね。」→ **はい**＝取った（止めるには信頼、認めるには不信のダブルスタンダード）。
- **Q2**「一人の医師より、複数の独立した医師が診断するほうが、誤診の危険は下がりますね。」→ **下がる**＝取った。
- **Q3**（余力で）「取れない苦痛のなかで末期患者を生かし続けることも、後戻りはできませんね。」→ **はい**＝不可逆は両刃。
- **落とし所**：同じ診断を尊厳死では信頼／複数医師で極小化／不可逆は双方向。

### 対・論点3（費用削減の危険）
- **Q1**「我方プランは経済的事情を要件から明文で除外しています。費用を理由に安楽死を選ばせることは、この制度のもとで可能ですか。」→ **できない**＝取った。
- **Q2**「我方は緩和ケアを尽くすことを義務づけます。これは医療費を、むしろ増やしますね。」→ **はい／増える**＝取った（"費用削減の近道"の逆）。
- **Q3**「医療費は社会全体の政策の問題です。安楽死を禁止すれば、その費用問題は解決しますか。」→ **しない**＝取った（別問題）。
- **落とし所**：費用は要件外／むしろ高コストを前置／費用問題は別途解決すべき。

### 対・論点4（医療資源の不足）
- **Q1**「我方の対象は『治療抵抗性の身体的苦痛』です。『医療機器を若い人に譲るべき』という理由は、この要件に当たりますか。」→ **当たらない**＝取った。
- **Q2**「『回復の見込みが低い』というだけで、我方の対象になりますか。」→ **ならない**＝取った（相手の前提を否定）。
- **Q3**「人工呼吸器が足りないなら、増やすか配分を見直すのが筋です。安楽死を禁止すれば、人工呼吸器は増えますか。」→ **増えない**＝取った（資源政策の誤爆）。
- **落とし所**：資源由来の動機は対象外／資源不足は別問題、禁止しても資源は増えない。

## 漏斗①｜「緩和ケア・鎮静で足りる」への攻め【予測・今回相手は未使用】

**否定の主張**：治療抵抗性の苦痛は持続的鎮静で対応できる。安楽死は不要。
**取りたい譲歩**：(a) 難治性の苦痛は実在する／(b) 鎮静を始め・続ける判断は本人ではなく医療チームがする＝本人は選べない。

- **Q1**「緩和ケアを尽くしてもなお取りきれない苦痛——治療抵抗性の苦痛——が、一部の患者に存在すること自体は、お認めになりますか。」
  - → **はい** ＝ メリット②の前提を取った。Q2へ。
  - → **いいえ** ＝「では、医療現場で行われている『持続的深鎮静』は、何のために存在するのですか。」（鎮静の存在そのものが、取りきれない苦痛の存在の証拠）
- **Q2**「その持続的鎮静を、いつ始め、どこまで続けるか。最終的に決めるのは、患者本人ですか、医療チームですか。」
  - → **医療チーム**（事実そう）＝ 取った。
- **Q3**「では、深く眠らせ続ける鎮静では、最期をどう迎えるか——意識を保つかどうか——を、患者本人は選べない。そうですね。」
  - → **はい** ＝ 取った。⚠️**時間の長短では争わない。「本人が選べるかどうか」で止める。**
- **落とし所**：難治性苦痛は実在（②）＋鎮静は本人が選べない（①）。両方を相手の口から裏づけさせた。

## 漏斗②｜「生命は不可侵」への攻め＋尊厳死との一貫性【常備】

**否定の主張**：生命は絶対に不可侵。
**取りたい譲歩**：(a) 生命は「絶対」不可侵ではない（本人意思で死を選ぶ余地を否定側も認める）／(b) 尊厳死では救えない患者がいる。

- **Q1**「患者が延命治療を拒否すること——治療をやめて、自然な死を選ぶこと——は、否定側もお認めになりますか。」
  - → **はい**（尊厳死は広く認められている）＝ Q2へ。
  - → **いいえ** ＝「では、本人が拒否しても、人工呼吸器を強制的につけ続けるべきだと。」（極端な立場に追い込む。以後の漏斗④もここに合流）
- **Q2**「では、生命は『絶対に不可侵』なのではなく、一定の場合には本人の意思で死を選ぶことを、否定側も認めている。そうですね。」
  - → **はい** ＝ 取った（生命絶対主義の放棄）。
- **Q3（決め手・空白の露呈）**「その延命治療の拒否は、人工呼吸器のような『止められる治療』がある患者しか救えません。膵臓がんで、止めるべき延命治療がないのに激痛に苦しむ患者を、尊厳死は救えますか。」
  - → **いいえ** ＝ 取った（尊厳死では埋まらない空白＝プランの必要性）。
- **落とし所**：生命は絶対不可侵ではない（相手が自己決定の余地を認めた）＋尊厳死では救えない患者がいる。

## 漏斗③｜「すべり坂（適用拡大）」への攻め

**否定の主張**：オランダのように対象が必ず拡大する。
**取りたい譲歩**：拡大は「不可避」ではなく「立法設計しだい」。

- **Q1**「否定側が懸念する『拡大』とは、精神疾患や認知症にまで広がった、オランダの事例ですね。」
  - → **はい** ＝ 拡大の中身を精神／認知症に固定。Q2へ。
- **Q2**「私たちのプランは、その精神的苦痛と、判断能力を欠く認知症を、法律の条文で対象から外しています。ご存じですか。」
  - → 知っていても知らなくてもQ3へ（防火墙の存在を場に出す）。
- **Q3**「オレゴン州は、施行から二十五年以上、対象を一度も拡大していません。事実として、ご存じですか。」
  - → **知らない** ＝ 相手の「必ず拡大」論の証拠不足を露呈。**知っている**＝Q4へ。
- **Q4**「では、拡大は『必ず起きる』のではなく、『法律でどう線を引くか』で決まる。お認めになりますね。」
  - → **はい** ＝ 取った。
- **落とし所**：すべり坂は不可避ではなく立法設計しだい＝比較衡量の「**防ぎうる**将来のリスク」を相手に認めさせる。

## 漏斗④｜「弱者への圧力」への攻め

**否定の主張**：家族への遠慮・圧力で、本心でない死へ追い込まれる。
**取りたい譲歩**：圧力は「禁止」の理由ではなく「歯止めの設計」で対処すべき問題。

- **Q1**「『家族に迷惑をかけたくない』という圧力の問題は、すでに認められている尊厳死——延命治療の中止——でも、同じように起こりえますね。」
  - → **はい** ＝ Q2へ。
  - → **いいえ**（尊厳死では起きない）＝「家族が中止を望む場面で圧力が働かない、と言い切れますか。」（東海大事件はまさに家族が中止を迫った事案）
- **Q2**「それでも否定側は尊厳死を認める。つまり、圧力の危険は『禁止』ではなく『歯止めの設計』で対処すべき問題だ。そうですね。」
  - → **はい** ＝ 取った。
- **落とし所**：圧力＝禁止理由にならない。我方は繰り返しの意思確認・代理決定の禁止・公的立会人・経済的事由の除外で対処。
- ※否定が尊厳死も認めない立場なら → 漏斗②Q1の「強制的に延命」へ合流させる。

## 漏斗⑤｜「誤診」「医師の役割」への攻め（軽め・余力があれば）

**取りたい譲歩**：誤診は複数医師で最小化できる／医師の役割は救命一辺倒ではない。

- **Q1**「『治る見込みがない』という診断は、現在の医療でも、治療方針を決めるために日常的に行われていますね。」→ **はい**
- **Q2**「複数の医師が独立して診断すれば、一人の判断よりも、誤診の危険は下がりますね。」→ **下がる**
- **Q3（医師の役割）**「医師は、現在でも、延命治療を中止して死を早めることが法的に認められています。では医師の役割には、『苦痛を和らげ、患者の意思を尊重すること』も含まれる。そうですね。」→ **はい**
- **落とし所**：誤診は複数医師で最小化／医師の使命は「いかなる場合も死なせない」ではない。

## 本番での回し方（中文）

- **质疑通常只有2〜3分钟**，跑不完5条。**先上能砸中否定1中心论点的那条**（对方主打"缓和ケアで足りる"就上①，主打"生命不可侵"就上②，主打"滑坡"就上③）。
- **②（尊厳死の一貫性）几乎万用**，强烈建议每场都留着——只要对方承认尊厳死/治療拒否，①生命非绝对②有尊厳死救不了的人，两个让步到手。
- **取って、黙ってしまう**。当场别展开论证（那是反駁的活）。每拿到一个"はい"，记进フロー、心里标记「これは反駁で使う」。
- **对方演说就打断**：「恐れ入ります、はい か いいえ で。」控场本身也算分（采分④・⑥）。
- **绝不问开放问题**（「なぜ」「どう思いますか」=送对方演讲时间）。每个问题的"对我有利的答案"必须是可预测的。
- 这套漏斗 **环节8（肯定2→否定2）也能用**，全队对一遍口径。
